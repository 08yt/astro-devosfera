之前我们聊过了使用 **Cloudflare Tunnel** 来实现家里群晖、甚至旁路由等 Web 面板的完美公网映射。

但在 2024 年，我们还会遇到很多特殊的临时场景。例如你只是想临时在电脑上给远在另一个城市的朋友展示你本地 `localhost:8080` 正在跑着的一个前置开发网页；又或者你是想要把自己公司里因为极其无理的安全网关限制而碰不到的一台极其深的开发机 `mysql` 端口，拉到你自己家里进行排查调试。

为了这么一件一次性的“小事”，去专门申请一串巨长极其冗长复杂的 Token 并再装一个 Cloudflare Tunnels 或者是神鬼莫测的 FRP 内网穿透服务，显然是大炮打蚊子、杀鸡用牛刀了。

这个时候，只要你手里有一台具有公网 IP 的跳板小鸡（VPS），并且你的终端自带了这地球上最伟大、也是每一秒都在被全世界开发者使用的工具 —— **SSH (Secure Shell)**，这就够了！
