手机作为我们上网的刚需设备，Sing-box 在移动端的表现可谓是“遥遥领先”。不仅因为它是使用 Go 语言的高效代码使得电池消耗极低，而且它的图形化体验也是原生打造的。

## Table of contents

## iOS 苹果端：SFI 的终极体验

对于 iPhone 用户来说，SFI (Sing-box for iOS) 目前是你能获取到的最好的官方客户端之一。

### 如何下载 SFI？
由于特殊的政策原因，SFI 不能上架中国大陆的 App Store。
**解决方案**：你需要一个“非国区”的女娲补天账号（例如美区、日区或港区 Apple ID）。登录该区域商店后，搜索 **"Sing-box"**，认准由 SagerNet 发布的蓝色图标即可免费下载。

### 导入你的节点
由于手机打字编辑长串 JSON 配置很难受，推荐你使用 **“Remote Profile（远程配置）”**。你的机场如果支持 Sing-box 订阅订阅，你可以直接填写这个订阅链接。SFI 会自动拉取服务器下发的配置文件。

### 开启与关闭
点击底部大大的“开始图标”，系统会提示你需要安装一个 VPN 描述文件（这叫 TUN 模式）。安装授权并允许后，你可以看到状态栏出现绿色的 VPN 图标。这说明一切就绪了。

## Android 平台：SFA (Sing-box for Android)

安卓系统的生态比较开放。通过 Github Release 你可以拉取到官方编译的。APK 安装包。
SFA 的最大亮点在于系统赋予的神级功能：**真身·分应用代理** (Per-App Proxy)。相比起还要自己写规则过滤软件流量，这是最有效杜绝中国大陆本地 App 走代理的手段。

### 安装与基础设置
1. 去 GitHub 最新版本中下载 `SFA-armeabi-v7a` 或者 `SFA-arm64-v8a`，安装到手机上。
2. 在 Profile 面板导入订阅或复制粘贴配置文件。

### 分应用代理配置教程
**为什么要分应用？** 
有些视频类应用（比如哔哩哔哩、爱奇艺或者抖音），它们有超大流量，如果被发送给节点不仅极度浪费，甚至会降速或异常。如果你只希望“推特”或“YouTube”经过代理，那就需要分应用设置。

1. 进入 SFA 的 **Settings** (设置页面)。
2. 在 **Route** 选项中，找到并开启 **Per-App Proxy (应用级别的分流代理)**。
3. 它会列出设备安装的所有应用列表：
   - 如果你想**仅代理部分**，可以选择 “Proxy Mode (白名单模式)”，勾选你想要的海外软件。
   - 如果你想**过滤少部分应用**，选择 “Bypass Mode (黑名单模式)”，并勾选支付宝、微信、爱奇艺等国内大厂全家桶。
4. 保存后重新连接节点即可生效。

## 结语
不管是 iOS 还是 Android，现在 Sing-box 的操作体验几乎已经是傻瓜式的一键操作。一旦你的配置设置好，在手机上几乎不需要经常切换节点。
享受极速丝滑和低功耗的上网生活吧！
